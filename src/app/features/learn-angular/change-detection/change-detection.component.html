Angular Change Detection

Change Detection

Change Detection is the mechanism Angular uses to update the UI whenever application data changes.

What is Change Detection?

Angular automatically checks component data and updates the DOM whenever a value changes. 

{{ username }}

How Change Detection Works

Angular runs change detection when:

Button Click HTTP Response Timer / setTimeout User Input

Default Change Detection

import { Component } from '@angular/core';

@Component({
  selector: 'app-user',
  templateUrl: './user.component.html'
})

export class UserComponent {
  name = 'John';
}

OnPush Change Detection

OnPush improves performance by checking changes only when input reference changes. 

import { ChangeDetectionStrategy, Component } from '@angular/core';

@Component({
  selector: 'app-user',
  changeDetection: ChangeDetectionStrategy.OnPush,
  template: '{{ name }}'
})

export class UserComponent {
  name = 'John';
}
