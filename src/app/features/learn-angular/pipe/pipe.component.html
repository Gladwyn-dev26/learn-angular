Angular Pipes

Pipes in Angular

Pipes are used to transform data in Angular templates. They help format values like dates, currency, numbers, and text before displaying them in the UI.

What is a Pipe?

A pipe takes input data and transforms it into a desired format directly inside the template. 


{{ username | uppercase }}

Built-in Pipes

uppercase lowercase date currency percent json

Pipe Example


{{ today | date:'fullDate' }}

{{ price | currency:'USD' }}

{{ name | uppercase }}

Custom Pipe

Developers can create custom pipes to transform data based on application needs. 


@Pipe({
 name: 'capitalize'
})

export class CapitalizePipe implements PipeTransform {

 transform(value: string) {
   return value.charAt(0).toUpperCase() + value.slice(1);
 }

}
