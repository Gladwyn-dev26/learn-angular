Angular Dependency Injection

Dependency Injection (DI)

Dependency Injection is a design pattern in Angular used to provide services to components instead of creating them manually.

What is Dependency Injection?

Angular automatically creates and injects services into components using its dependency injection system. 


constructor(private userService: UserService) {}

Why Dependency Injection?

Loose Coupling Reusable Services Easy Testing Better Maintainability

Service Example


import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class UserService {

  getUsers() {
    return ['John', 'David', 'Sara'];
  }

}

Injecting Service into Component


constructor(private userService: UserService) {

  const users = this.userService.getUsers();

  console.log(users);

}
