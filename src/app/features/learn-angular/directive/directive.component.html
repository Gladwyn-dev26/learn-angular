Angular Directives

Directives in Angular

Directives are classes that modify the behavior or appearance of DOM elements. They allow developers to extend HTML with custom behavior.

What is a Directive?

A Directive is a TypeScript class that adds behavior to elements in the DOM. Angular uses directives to manipulate the DOM and control UI behavior.

Types of Directives

Component Directive Structural Directive Attribute Directive

Structural Directives

Structural directives change the structure of the DOM by adding or removing elements.

*ngIf   *ngFor   *ngSwitch

Attribute Directives

Attribute directives change the appearance or behavior of an element.

ngClass   ngStyle

Custom Directive Syntax


@Directive({
 selector: '[appHighlight]'
})

export class HighlightDirective {

 constructor(private el: ElementRef) {
   this.el.nativeElement.style.backgroundColor = 'yellow';
 }

}
